# Transmissao-som-pela-luz

Este projeto tem como objetivo transmitir o sinal analógico de uma guitarra utilizando comunicação óptica via laser.
O áudio captado pela guitarra é convertido em modulação luminosa, transmitido por um feixe laser e recebido por um sensor óptico desenvolvido pelos autores.
Após o tratamento eletrônico do sinal recebido, o áudio é reproduzido em uma caixa de som.

## Objetivos

- Transmitir áudio utilizando comunicação óptica
- Implementar tratamento de sinal no receptor

## Funcionamento

1. O sinal analógico da guitarra é inserido em uma lazer
2. O laser é modulado conforme o áudio
3. O feixe é transmitido até o LED receptor (fotodiodo)
4. O sinal recebido passa por filtragem e amplificação
5. O áudio é enviado para a caixa de som
